## About

Computer engineer with hands-on experience in infrastructure, technical support, and hardware. I've worked from server rooms to classrooms — currently teaching robotics, app development, and game design at public schools in Queimados, RJ, while building IoT projects, SaaS tools, and homelab stacks on the side.

I like systems that are well-documented, problems that have root causes, and solutions that actually last. If it breaks, I want to know why.
